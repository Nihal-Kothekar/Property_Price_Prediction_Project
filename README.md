# Property Price Prediction Streamlit Dashboard

This is an interactive Streamlit dashboard for the Property Price Prediction project.

## Files needed

Keep these files in the same folder:

- app.py
- property_price_data.csv
- requirements.txt

## How to run

Open terminal or Anaconda Prompt in the project folder and run:

```bash
pip install -r requirements.txt
streamlit run app.py
```

The dashboard will open in the browser.

## Dashboard sections

- Home
- Data Overview
- EDA Charts
- Missing Values
- Model Performance
- Predict Price
- Insights
